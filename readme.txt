This plugin adds a simply footer panel like the one on facebook.

== Description ==

This plugin was developed by Janvier at <a href="http://www.wprooms.com">Wp rooms</a> based on Soh tanaka's facebook style footer adimn panel at http://www.wprooms.com/2010/01/05/facebook-like-foot-panel.html

http://forums.wprooms.com/topic/facebook-footer-panel

It adds a footer panel like the one on facebook for buddypress. 

### Known Issues ###

*jquery conflict if there are more than one plugin calling it.
*Wide page CSS errors (to be fixed in next version)

== Installation ==


1. Upload the entire facebook-foot-panel directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
